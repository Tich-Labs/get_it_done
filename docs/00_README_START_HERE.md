### Common Questions Answered

**Q: Can we really build this in 2 weeks?**
A: Yes. The MVP scope is tight: profile + services + availability + booking + payment. No fancy analytics, no integrations, no mobile app.

**Q: Is zero-cost hosting really possible?**
A: Yes. Vercel (100GB compute/month free), Supabase (500MB storage free), SendGrid (100 emails/day free). All included in costs.

**Q: How do we handle M-Pesa payments?**
A: Via Pesapal (2.5% transaction fee). MVP uses hosted link, not custom integration. Simplest, fastest path.

**Q: What if Calendly or bigger players enter Kenya?**
A: We have community moat (early users build network effects) + Kenya-first approach (they won't optimize for local market) + price advantage (KES 500 vs their $15-20).

**Q: When do we hire?**
A: Post-launch. MVP built by 1 full-stack + 1 PM. Month 2+: add growth person. Month 3+: consider support person if needed.
